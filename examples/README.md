### 运行需要

* [JDK 21+](https://www.oracle.com/java/technologies/downloads/)


### 下载

* [lealone-8.0.0-SNAPSHOT.jar](https://lealone-plugins.github.io/lealone.github.io/lealone/lealone-8.0.0-SNAPSHOT.jar)


## 下载 AI 应用

`git clone https://github.com/lealone/Lealone-Docs.git lealone-examples`


### 简单 AI 应用

先替换 hello/services.sql 中的 api_key

然后执行 `java -jar lealone-8.0.0-SNAPSHOT.jar hello`

在浏览器中打开: http://localhost:8080/hello.html


### 全栈 AI 应用

先替换 crud/services.sql 中的 api_key

然后执行 `java -jar lealone-8.0.0-SNAPSHOT.jar crud`

在浏览器中打开: http://localhost:8080/crud.html


### 经典的 petstore 应用

先替换 petstore/llm.sql 中的 api_key

然后执行 `java -jar lealone-8.0.0-SNAPSHOT.jar petstore`

在浏览器中打开: http://localhost:8080/
